# Recipes

Short, focused examples that build on the [Quickstart](../quickstart/connect-to-an-indexer.md). Each assumes you have a connected SDK instance.

| Recipe | Description |
|--------|-------------|
| [Upload Packing](upload-packing.md) | Batch many small objects into a single upload session |
| [Upload from a File](upload-from-file.md) | Stream directly from disk instead of memory |
| [Resume a Download](resume-download.md) | Continue a partial download using byte offsets |
| [Download to a File](download-to-file.md) | Stream decrypted bytes directly to disk |
| [Object Metadata](object-metadata.md) | Attach or update encrypted application metadata |
| [Share an Object](share-an-object.md) | Generate public, time-limited download URLs |
| [Delete and Prune](delete-and-prune.md) | Remove objects and clean up unreferenced slabs |
