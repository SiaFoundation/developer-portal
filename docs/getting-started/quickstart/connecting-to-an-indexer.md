# Connecting to an Indexer

Before your app can upload, download, or share data with Sia, it must first connect to an indexer. An indexer acts as your application’s gateway to the Sia network. It handles:

  * **Authentication** - Verifying your app key.
  * **Authorization** - Ensuring the user approves your app.
  * **Metadata tracking** - Storing references to pinned objects.
  * **Host coordination** - Telling the SDK which storage providers to use.

## Prerequisits

Before starting, make sure you have:

* **A valid indexer URL** - This guide will use `https://app.sia.storage`, but any indexer will work.
* **A 32-byte App ID** - Provided to you by your indexer of choice.
* **The SDK installed for your language** - See [Getting Started](../README.md) for instructions.

## Authentication Requirements

Each new instance of your app will require a unique App Key which is deterministically derived from:

* **A BIP-39 recovery phrase** - Generated by user and acts as their app login
* **A 32-byte App ID** - A unique 32-byte identifier for your app. The value is not secret, but it should be random and unique to the app.
!!! warning
    The App ID must remain consistent for your application.

    * Generate it once per app, not per user or per device.
    * Keep it stable across updates or installations.
    * Changing the App ID changes the derived app key.

The App Key is how your application will uniquely identify itself to the indexer.

### Connection flow

Connecting to an indexer happens in three steps:

1. **Create an App Key:** This is your app’s identity.
2. **Initialize the SDK:** Bind your App Key to the indexer you want to use.
3. **Request User Approval:** The indexer provides a URL the user must visit to approve your app. *(First run only)*

Once the app is approved, the SDK connects silently on future launches.

## Example

=== "Python"
    ```python
    import asyncio
    from indexd_ffi import (
        generate_recovery_phrase,
        AppKey,
        AppMeta,
        Sdk,
    )

    async def main():
        # 1. Create or restore your app key
        seed_phrase = generate_recovery_phrase()
        app_id = b'\x01' * 32
        key = AppKey(seed_phrase, app_id)

        # 2. Initialize the SDK
        sdk = Sdk("https://app.sia.storage", key)

        # 3. Connect / request approval
        if not await(sdk.connected()):
            meta = AppMeta(
                name="My App",
                description="Demo application",
                service_url="https://example.com",
                logo_url=None,
                callback_url=None
            )
            resp = await(sdk.request_app_connection(meta))

            print("Open this URL to approve the app:", resp.response_url)

            approved = await(sdk.wait_for_connect(resp))
            if not approved:
                raise Exception("User rejected the app")

        print("Connected!")

    asyncio.run(main())
    ```
