# Troubleshooting & Common Errors

| Error                      | Explanation                                                  |
| -------------------------- | ------------------------------------------------------------ |
| **Invalid app key length** | App ID must be exactly 32 bytes.                             |
| **App not approved**       | User rejected the approval request or never opened the link. |
| **Network / QUIC errors**  | Check connectivity to the indexer.                           |
| **Object not found**       | Share URL may be expired or incorrect.                       |
